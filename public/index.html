Canvas Lock

Lock button in HUD disables horizontal panning. Prevents accidental page movement when writing with zoom enabled.

Toolbar Sizing

Configurable toolbar sizes: Small, Medium, Large. Adjust UI elements to match your screen and preferences.

Vertical Scrollbar

Draggable scrollbar on the right side. Navigate through long notes quickly without repeated swipe gestures.

Telemetry Disabled

Sentry telemetry completely disabled. Fake DSN replaces official endpoint. Zero external data transmission.

Parallel Installation

Package ID changed to com.adilhanney.saber.fork. Install alongside original Saber without conflicts.

Stylus Pointer

Visual pointer shows stylus position when hovering. Color matches selected pen, square shape for eraser. Hides while writing, reappears after 1 second.

Enhanced Palm Rejection

Improved palm rejection activates when stylus is detected near screen. Prevents accidental touches while writing, even when palm touches screen.

Performance Optimizations

Separated stroke rendering layers for smoother writing. Low-quality paths during drawing, high-quality when complete. Reduced unnecessary redraws for better performance.
